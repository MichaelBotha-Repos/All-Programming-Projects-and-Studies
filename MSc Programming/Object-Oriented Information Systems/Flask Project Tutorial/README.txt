> This is a tutorial I completed found at https://flask.palletsprojects.com/en/2.0.x/tutorial/tests/
> It is a basic Flask application which allows one to create a profile, login, and create a blog post
> It uses the SQLite package which is native to the standard Python libraries to create a database
> One would run the application by downloading "Flask Project Tutorial" folder, starting a command prompt
session on your device, changing the directory to the "Flask Project Tutorial" folder and entering the 
following commands:
1.) set FLASK_APP=ApplicationCodeAndFiles
2.) set FLASK_APP=development
3.) flask init-db
4.) flask run
